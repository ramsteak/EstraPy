# Examples

This section collects complete, practical EstraPy workflows for common XAS analysis tasks.
Each example shows a realistic command sequence, recommended parameter choices, and expected outputs.

The current examples are:

1. **[XANES Example]({{ "/examples/xanes-example" | relative_url }})** - End-to-end XANES-oriented workflow
2. **[EXAFS Example]({{ "/examples/exafs-example" | relative_url }})** - End-to-end EXAFS workflow including background and Fourier analysis

## What You Will Find in Each Example

Each example page is structured to be copy-paste friendly and includes:

- Input assumptions and required files
- Step-by-step EstraPy commands
- Brief explanation of each processing stage
- Suggested plotting/check steps
- Export/saving step for final data

## How to Use These Examples

1. Start from the example that matches your analysis goal (XANES or EXAFS).
2. Run the commands as provided on your dataset.
3. Adjust ranges and options to your edge, element, and data quality.
4. Re-run and compare plots after each major step.
